概述

给代理工作者： 必须使用的子技能：使用 superpowers:executing-plans 逐任务实施此计划。步骤使用复选框（- [ ]）语法进行追踪。

目标： 在行情页面左侧添加三个（A股/港股/美股）可自动伸缩的垂直侧边栏，收缩时显示市场标签，展开时显示完整指数数据和迷你走势图，三个侧边栏水平排列且不影响主内容布局稳定性。

架构： 重用现有 MarketSidebar.vue 组件（已具备伸缩、ECharts sparkline、点击外部收起等功能），将其集成到 MarketView.vue 页面中。页面布局采用 flex 双栏结构：左侧为侧边栏条带（三个 MarketSidebar 水平排列），右侧为主内容区（指数卡片 + 温度计 + 涨跌排行）。侧边栏展开时通过 flex-shrink: 0 和固定宽度确保主内容区不被挤压。

技术栈： Vue 3 Composition API + Pinia + ECharts + CSS Variables + Flexbox

// 在 setChartRef 函数中，chart.setOption(...) 之后添加：
const ro = new ResizeObserver(() => {
  chart.resize()
})
ro.observe(el_)
// 存储 ResizeObserver 以便清理
resizeObservers.set(code, ro)
// 在 chartInstances 声明之后添加：
const resizeObservers = new Map<string, ResizeObserver>()
function disposeAllCharts() {
  for (const [, ro] of resizeObservers) {
    ro.disconnect()
  }
  resizeObservers.clear()
  for (const [, chart] of chartInstances) {
    chart.dispose()
  }
  chartInstances.clear()
}

  • 需求1（自动伸缩 + 点击展开/收起 + 滚轮滚动）→ MarketSidebar 已有此功能，任务1优化 resize
