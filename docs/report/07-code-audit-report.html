  • C-1 StockHeader 自选功能调用了错误的 store 方法 — 股票被当作基金添加到自选列表，用户可直接感知的功能 Bug
  • C-2 股票预测/行情 API 层全面使用 any 类型 — 从类型定义到 API 层到 store，类型安全完全失守
  • C-3 FundDetailService.cache 并发读写无保护 — map 并发读写将导致 fatal error: concurrent map read and map write，进程崩溃

composables/useECharts.ts 行 1-45

CRITICAL StockKline 周K/月K 聚合逻辑存在数据偏差 ✅ 已修复 FE-C04
