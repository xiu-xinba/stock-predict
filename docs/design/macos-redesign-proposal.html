Design Redesign Proposal

macOS 风格重设计方案

基于对当前界面的全面审计，提出四种 macOS 设计语言方向的重设计方案，覆盖色彩、排版、间距、材质、动效和交互六个维度。

Current State Audit

当前界面问题诊断

通过系统性审查，当前界面存在以下核心问题，导致视觉品质低于 macOS 原生应用水准。

🎨

色彩体系偏冷偏荧光

暗色模式背景 #0b0f19 偏蓝黑而非中性灰，涨跌色 #ff8a7a / #47d8a0 在暗色下过于荧光，品牌蓝 #175cd3 饱和度过高。

📐

排版阶梯模糊

10/11/12px 三级字号仅差 1px，视觉上无法区分层级。中文在 10-11px 下几乎不可读，行高 1.1 对中文排版过于紧凑。

🪟

缺少 macOS 材质感

无 vibrancy 分层效果，topbar/dock 的 backdrop-filter 过于简单，缺少 saturate() 和顶部高光边框，卡片无内阴影深度感。

📏

间距缺乏节奏

所有卡片使用相同的 card-accent-top 样式，摘要与列表无视觉权重差异。metric 间使用硬边框分割而非间距呼吸。

🔲

圆角与阴影偏保守

卡片圆角 12px 低于 macOS 标准的 14-16px，阴影透明度仅 0.04-0.06 几乎不可见，无法产生浮起感。

⚡

交互反馈不一致

不可点击元素有 hover 效果，可点击元素缺 active 态，折叠图标用 Unicode 字符而非 SVG，Dock 缺少鱼眼放大动画。

Design Principles

macOS 设计原则

所有方案共享以下六条 macOS 设计核心原则，确保重设计方向的一致性。

01

材质分层

通过背景色层级和 vibrancy 效果建立深度，而非依赖边框和阴影。window / content / elevated 三层材质系统。

02

排版即层级

字号阶梯间保持 1.25 倍以上比例，最小中文字号不低于 12px。通过字重和字间距辅助层级区分。

03

克制用色

品牌色占界面面积不超过 10%，仅用于交互元素和关键语义。中性色偏暖灰调，避免冷蓝底色。

04

微妙深度

阴影使用三层叠加（边框阴影 + 中距投影 + 远距投影），暗色模式用 glow 替代 shadow。圆角 14-16px。

05

动效服务状态

动画仅用于状态转换反馈，使用 ease-out 曲线。禁止 bounce/elastic。尊重 prefers-reduced-motion。

06

原生交互模式

Dock 鱼眼放大、列表 hover 高亮、卡片折叠 chevron 旋转、按钮 active 缩放。遵循平台习惯。

Shared Changes

所有方案共享的改动

以下改动不依赖方案选择，是所有 macOS 风格重设计的共同基础。

排版重构

  • 合并 10/11/12px 为 12px 一个层级
  • 字号阶梯保持 1.25x 以上比例
  • 最小中文字号不低于 12px
  • 行高最低 1.3（中文排版）
  • 标题添加 letter-spacing: -0.01em
  • 小标签添加 letter-spacing: 0.02em

交互统一

  • 折叠图标改为 SVG chevron + 旋转动画
  • 可点击元素添加 active 态缩放
  • 移除非交互元素的 hover 效果
  • 列表 hover 使用背景色变化而非位移
  • Dock 添加鱼眼放大动画
  • Topbar 按钮统一为 28-32px

间距节奏

  • 摘要区与列表区视觉权重区分
  • metric 间使用 gap 替代硬边框
  • fund-row 上下间距增加至 12px
  • fund-code/fund-name 间距增加至 4px
  • 排行面板间距缩至 16px
  • 骨架屏与实际布局对齐

视觉层级

  • card-accent-top 2px 线改为 1px 渐变线
  • fund-row 方向指示条增至 3px
  • 排名数字使用圆形 + 更深背景
  • 空状态增大插图和 CTA 按钮
  • 价格区字号增至 36-48px
  • kv-label 字号增至 12px
Recommendation

推荐方案

基于 PRODUCT.md 定义的品牌个性（冷静、专业、可靠）和用户场景（通勤/午休/睡前快速查看），推荐以下方案组合。

首选：A. Sonoma 深色

最贴合"冷静、专业、可靠"的品牌调性。中性暖灰底色沉稳不压抑，品牌蓝克制使用，涨跌色降饱和度避免情绪刺激。三层材质系统通过背景色差异建立深度，实现复杂度低，性能开销小。与当前暗色优先的设计方向一致，迁移成本最低。

品牌调性匹配 实现成本低 性能友好 迁移成本最低

备选：D. 暖中性

如果希望跳出金融 App 冷蓝调的"第一训练数据反射"，暖中性方案提供独特的视觉辨识度。琥珀色强调远离行业套路，暖棕灰底色营造沉稳可信赖的氛围。实现复杂度与 Sonoma 相当，但需要更多设计勇气。

独特辨识度 实现成本低 远离金融套路 需设计勇气

不推荐 C. Sequoia 玻璃态作为首选的原因

虽然视觉冲击力最强，但大面积 backdrop-filter 在低端设备和移动端有显著性能开销。PRODUCT.md 的用户场景包含"通勤地铁上单手操作手机"，移动端性能和可读性优先级高于视觉炫技。此外，玻璃态与"克制即尊重"的设计原则存在张力。可作为后续迭代的视觉增强选项，在桌面端渐进启用。
