一、项目整体架构与功能模块

1.1 架构设计

本项目是一个基于机器学习的基金净值预测系统，采用前后端分离 + 独立模型训练服务的三层架构：

前端 (Vue3)
Vite + Pinia
ECharts + Element Plus
REST API →
后端 (Go/Gin)
内存存储 + 代理
基线预测降级
HTTP →
模型服务 (Python)
joblib 推理服务
锦标赛 Champion

核心设计思路

  • 模型与后端解耦：Python 模型服务独立部署，Go 后端通过3个环境变量代理调用，不可用时自动回退 Go 基线预测
  • 长短周期分离：日级/周级预测与日内3/5分钟预测使用不同模型族、不同特征窗口
  • Champion/Challenger 机制：6候选模型锦标赛择优，新模型必须通过门槛才能晋升 Champion
